### Check if a Number is Perfect

A perfect number equals the **sum of its divisors** (excluding itself).

Example:

`6`

```
1 + 2 + 3 = 6
```

So **6 is a perfect number**.

### Check if a Number is an Armstrong Number

Example:

`153`

```

```

So it is an **Armstrong number**.
